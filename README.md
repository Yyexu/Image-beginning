# 图像处理任务 | 学习清单 | 2025.11 - 2025.12
## 1. 图像灰度化与通道分离  
题目描述： 编写程序，读取一张彩色图像，并实现以下功能：  
将其转换为灰度图像（使用加权平均法：Gray = 0.299*R + 0.587*G + 0.114*B）。  
将其 RGB 三个颜色通道分离，并分别显示。  
技术点： 图像读取与显示、颜色空间、数组操作。  
考察目标： 理解彩色图像的构成和像素级的操作。  

## 2. 图像几何变换
题目描述： 不直接使用库中现成的 warpAffine 函数，手动实现图像的以下变换：  
平移。  
绕图像中心的旋转。  
缩放（最近邻插值和双线性插值）。  
技术点： 仿射变换、前向映射与反向映射、插值算法。  
考察目标： 深入理解几何变换的数学原理和插值的重要性。  

## 3. 直方图均衡化
题目描述： 编写程序，实现灰度图像的直方图均衡化。分析并比较处理前后图像的直方图分布和视觉效果。  
技术点： 灰度直方图、累积分布函数、图像增强。  
考察目标： 掌握利用直方图进行图像增强的基本方法。  

## 4. 空间滤波器的实现与应用
题目描述：手动实现均值滤波、高斯滤波和中值滤波算法。  
用这些滤波器处理含有高斯噪声和椒盐噪声的图像，比较它们的去噪效果。  
技术点： 卷积操作、线性与非线性滤波器、噪声模型。  
考察目标： 理解不同滤波器的原理、优缺点及适用场景。  

## 5. 边缘检测综合实验
题目描述： 对同一张图像，分别实现并应用以下边缘检测算子：  
Sobel 算子  
Prewitt 算子  
Laplacian 算子  
Canny 算子（包括非极大值抑制和双阈值连接）  
技术点： 图像梯度、一阶与二阶微分算子、边缘连接。  
考察目标： 比较不同边缘检测算子的效果，深入理解 Canny 算子的优越性。  

## 6. 基于阈值的图像分割
题目描述： 实现以下阈值分割方法，并用于分割前景和背景：  
全局固定阈值（如 Otsu 法）。  
自适应阈值。  
技术点： 阈值选择、大津法、局部统计。  
考察目标： 掌握基本的区域分割方法，理解全局与局部阈值的适用性。  

## 7. 基于区域生长的图像分割
题目描述： 实现一个简单的区域生长算法。用户通过鼠标点击选择一个种子点，程序根据  
像素灰度相似性（例如，与种子点的差值小于某个阈值）生长出一个连通区域。  
技术点： 区域生长、种子点、相似性准则、队列/栈数据结构。  
考察目标： 理解一种经典的区域分割算法及其实现细节。  

四、 深度学习与高级应用
这些题目结合了现代深度学习方法。

## 8. 基于 CNN 的图像分类  
题目描述： 使用 PyTorch 或 TensorFlow 框架，在一个小型标准数据集（如 CIFAR-10）上完成以下任务：  
搭建并训练一个简单的卷积神经网络（CNN）进行图像分类。  
使用数据增强技术（如随机翻转、旋转）来提高模型泛化能力。  
技术点： 卷积神经网络、全连接层、Softmax、数据增强、模型训练。  
考察目标： 掌握使用深度学习框架解决图像分类问题的基本流程。  

## 9. 图像风格迁移
题目描述： 实现一个简单的图像风格迁移程序。给定一张内容图像和一张风格图像（如梵高的画作），生成一张既保留内容图像结构又具有风格图像艺术风格的图片。  
技术点： 预训练 CNN（如 VGG）、内容损失、风格损失（Gram 矩阵）、梯度下降。  
考察目标： 理解如何利用深度神经网络的特征表示进行创造性的图像处理任务。  、

## 10. 图像超分辨率重建
题目描述： 构建一个模型（如 SRCNN - 超分辨率卷积神经网络），将低分辨率图像重建为高分辨率图像。可以自己制作训练数据（通过对高清图像进行下采样）。  
技术点： 图像超分辨率、全卷积网络、均方误差损失、PSNR/SSIM 评估指标。  
考察目标： 学习解决图像重建问题，并了解如何定量评估图像质量。   

## 综合项目
简易车牌识别系统
题目描述： 设计一个完整的流程，从一张包含车辆的图像中识别出车牌号码。流程可以包括：  
车牌定位： 利用颜色、边缘或纹理特征在图像中粗略定位车牌区域。  
车牌校正： 对倾斜的车牌进行透视校正。  
字符分割： 将车牌上的每个字符分割出来。  
字符识别： 使用模板匹配或训练一个简单的分类器来识别每个字符。  
技术点： 综合运用边缘检测、形态学操作、轮廓分析、模板匹配/机器学习。  
考察目标： 培养学生将多个图像处理模块组合起来解决一个复杂实际问题的能力。
